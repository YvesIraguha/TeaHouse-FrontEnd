# TeaHouse

A platform for art-works showcasing

## Getting Started

### Technologies

- React
- Axios
- Redux
- Redux-thunk
- CSS

### Setting up development environment

```
- yarn add
- yarn start
```

After starting the server, the server will open a browser from your local PC

## Running the tests

- `yarn run test` to run tests

### And coding style tests

[Airbnb](https://github.com/airbnb/javascript) style guide
